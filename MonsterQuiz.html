MONSTER QUIZ

This is a project I worked on with one other developer and 3 artists. We had to make a project that would visualize some kind of data. We could decide for ourselves what we wanted to do. Our goal was to show how different everyone is by showing the data through the visuals of monsters.

HOW IT WORKS

This game is a sort of quiz, you get questions which you answer and with those answers you get a different type of monster each time. The game collects your data and with this for each specific answer you can possbibly fill in there will be a different visual output. So for example there will be a question where you choose your eye color, which end up changing the eyes of the monster, giving each option a different output. There is a total of 12 questions, which makes it over 25.000.000 different possible monsters outcomes.
